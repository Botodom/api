# Botodom API

This is a API made for the __Botodom__ project.

## Functionalities

* Eligibility submissions
* Homework Management

*This feaftures will be implemented in a future app*

## Run the project

`pip install -r requirements.txt`

`cd api`

`uvicorn main:app --reload --host 0.0.0.0`
